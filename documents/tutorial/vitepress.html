搭建环境检测 ​

win+R终端打开cmd，输入node -v，请确保你的 Node.js 版本号 >= 16。

javascript
node -v

创建博客 ​

  • 安装 vitepress
javascript
npm install -D vitepress
  • 构建基本项目
javascript
npx vitepress init
  • 运行
javascript
npm run docs:dev
